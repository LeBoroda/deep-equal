# Проект для проведения глубокого сравнения объектов в JavaScript

## Технологии:

- JavaScript
- Node.js
- Jest (тестирование)
- Eslint + Prettier (линтеры)

## Как пользоваться проектом:

Проект запускается из консоли. Точка входа: index.ts

- Запуск проекта: npm start
- Запуск тестов: npm run test
- Запуск линтеров: npm run lint (проверка), npm run lint:fix (проверка и исправление ошибок)

Различные виды компараторов разнесены по разным классам для удобства поддерживания и независимости.

### DeepEqualEngine

Оркестрирует компараторами **deep-equal-engine**. При создании он инициализирует все подключенные компараторы и выбирает соответствующий пра вызове функции deepEqual().
В классе реализована проверка циклических ссылок в объектах.

### Сравнение типов

Проект позволяет сравнивать различные типа JS объектов:

- примитивы (number, string, boolean, null, undefined, symbol)
- массивы
- объекты
- даты

### Расширение функциональности

Возможно добавление других типов объектов для проведения сравнения.
Новые модули компараторов должны размещаться в отдельных файлах и подключаться в оркестраторе в блоке инициализации

**ВАЖНОЕ ПРИМЕЧАНИЕ**

Так как новые проверяемые объекты могут относиться к классу Object в JS, их компараторы должны всегда добавляться в начало списка компараторов, либо выше в очереди, чем компаратор объектов, чтобы предупредить некорректную работу при сравнении.

В модуле **index.ts** создается экземпляр класса **deep-equal-engine**, который принимает 2 аргумента и производит их сравнение.
